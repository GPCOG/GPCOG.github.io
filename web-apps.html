Introduction

Here are some Shiny web apps we use at GPCOG.

IRS Migration Dashboard

The IRS Migration Dashboard allows you to view IRS Migration Data data by county since 2011.

Fiscal Impact Tool

The Fiscal Impact Tool allows you to estimate the fiscal impact of different development patterns.

Workforce Estimation Tool

The Workforce Estimation Tools allows you to view how the size of the workforce changes in response to changes in working age population, workforce participation, and migration. This app was used to validate the Connect 2045 employment forecast.

Population Pyramid

This app uses the cohort survival model to project the current population age structure into the future.
